# Trebuchet
A "whipper" style trebuchet designed to be manufacturable from a single 300x620x15mm MDF board on a CNC mill with the least amount of machining time possible. The goal was to win a trebuchet competition against other teams with the same constraints. Note: ball bearings where not aloud!
